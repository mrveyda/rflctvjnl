# Journaling App Frontend

This is the static frontend for the Journaling App, built with HTML, CSS, and JavaScript.

## Files

- `index.html`: Home page
- `register.html`: User registration
- `login.html`: User login

## Setup

No setup required - it's static HTML/JS.

## API Integration

The frontend calls the backend API at `https://your-backend-url.onrender.com/api`.

Update the `API_BASE` variable in each HTML file with the actual backend URL.

## Authentication

Uses JWT tokens stored in localStorage.

## Deployment

Hosted on Render as a static site.
